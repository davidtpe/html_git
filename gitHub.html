Git + GitHub 指令

  1. git pull 下載更新
  2. git pull = git fetch + git merge
  3. Local 改完檔案 add + commit 之後，可以 git push 推上 GitHub主機
  4. Local 如果 git push 產生錯誤，則先 git pull --rebase (同步下來)之後再 git push
  5. git push -f 就是 --force(強制)，會強迫硬推上去，覆蓋 GitHub上面的版本
  6. git clone git@github.com:kaochenlong/dummy-git.git 複製一份回Local
  7. git clone git@github.com:kaochenlong/dummy-git.git 複製一份回Local
  8. git clone git@github.com:kaochenlong/dummy-git.git hello_kitty 克隆下來之後指定 hello_kitty目錄
