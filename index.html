重要 Git 指令

  1. Git init 該資料夾建立一個 Git Repository
  2. Git add index.html
  3. Git add --all 將該管理資料夾全部加入
  4. Git status 查看 Git 最新狀態
  5. Git commit -m "5/30更新"
  6. Gitk GUI工具
  7. Git config --global core.editor notepad
  8. Git commit 指令不帶 -m 的話，會指定editor
  9. Git diff index.html 指令
  10. Git log --pretty=oneline 檢視 git 歷史紀錄 LOG
  11. Git reset --hard HEAD^ 回復前一版本，HEAD^^就回到前前一個版本=HEAD~2
  12. Git reset --hard 3628164 回復 commit id是“3628164...”的版本
  13. Git reflog 紀錄你的每一次命令
  14. Git show HEAD 顯示最新 Commit 的詳細記錄
  15. Git add index.html 尚未 Git Commit 之前就後悔，直接下 git reset index.html可以取消 Git add 狀態
  16. Git checkout 3628164 -- index.html 針對 index.html這個文件，回復到commit id是“3628164...”的版本
