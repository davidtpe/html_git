重要 Git 指令

  • Git init 該資料夾建立一個 Git Repository
  • Git Add index.html
  • Git status 查看 Git 最新狀態
  • Git commit -m "5/30更新"
  • Gitk Git Tool Kit
  • Git config --global core.editor notepad
  • Git commit 指令不帶 -m 的話，會指定editor
  • git diff index.html 指令
  • git log --pretty=oneline 檢視 git 歷史紀錄 LOG
  • git reset --hard HEAD^ 回復前一版本
  • git reset --hard 3628164 回復 commit id是“3628164...”的版本
